# Памятка по публикации BEAUTY ATELIER

1. **GitHub**
   - Репозиторий не содержит секретов (env, пароли, токены).
   - .gitignore настроен: node_modules, .next, .env*, .vercel и др. не попадут в репозиторий.

2. **SEO**
   - robots.txt и sitemap.xml добавлены в public/.
   - Проверьте favicon (public/favicon.ico) и мета-теги (app/layout.tsx).

3. **Vercel**
   - Проект разворачивается через GitHub.
   - После деплоя — подключить домен через Vercel Dashboard.
   - DNS: укажите CNAME или A-записи, которые даст Vercel.
   - SSL (https) подключается автоматически.

4. **Проверки**
   - Откройте сайт на домене, проверьте все страницы и языки.
   - Проверьте мобильную версию и скорость загрузки.

5. **Рекомендации**
   - Добавьте сайт в Google Search Console и Яндекс.Вебмастер.
   - Не храните секретные ключи в публичном доступе.
   - Для переменных окружения используйте Vercel → Project Settings → Environment Variables.

6. **Контакты**
   - Если потребуется помощь — пишите вашему разработчику или в Telegram @dar_sofy.

Удачного запуска BEAUTY ATELIER!
